Node and express JS

Technically, Node JS is a wrapper on google chrome's V8 engine, which enables one to write servers in javascript. It is asynchronous and hence high performance. Express JS is a lightweight, unopinionated framework which allows different functionalities like logging, authorisation, url parsing via components called middleware. Node Js can also be used to develop CLI applications apart from web applications.

ABAP

ABAP is a fourth generation language used to develop SAP applications. It is completely object oriented and resembles Java and C++ in many aspects. CDS and BOBF are frameworks that allow development based on SAP S4 HANA Architecture. BOBF allows to prototype applications( backend applications) rapidly while CDS is used to generate ODATA services that can be consumed by SAP UI5 applications.(frontend applications).
